Internal

Se puede desplegar mensajes JS dentro de un archivo HTML, poniendo el tag <script> [código en JS] </script>, en esta oportunidad se ha puesto:

<script>

alert("Hello, world.");

for (var i = 0; i < 1; i++) {

alert("No te vayas de esta página, eres el visitante 9999")

if (i == 1) {

alert("Por favor!")

}

}

</script>

También se puede crear un archivo .js para poder controlar los objetos insertos en HTML.
